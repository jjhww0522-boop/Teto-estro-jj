# 카카오톡 공유 시 링크가 안 열릴 때

공유 카드를 눌렀을 때 **링크로 연결이 안 되면** 아래를 확인하세요.

---

## 1. 카카오 개발자 콘솔에 도메인 등록 (필수)

카카오는 **등록된 도메인**에서만 공유 링크가 정상 동작합니다.

1. [Kakao Developers](https://developers.kakao.com) 로그인
2. **내 애플리케이션** → 사용 중인 앱 선택
3. **앱 설정** → **플랫폼** → **Web** 플랫폼 추가/수정
4. **사이트 도메인**에 배포 주소 입력 후 저장  
   - 예: `https://teto-estro-jj-knyb.vercel.app`  
   - **도메인만** 입력 (끝에 `/` 없이, `https://` 포함)
5. Vercel 등으로 도메인을 바꾼 경우, 여기서도 같은 도메인으로 수정

등록하지 않으면 공유 카드를 눌러도 링크가 열리지 않을 수 있습니다.

---

## 2. 환경 변수 (선택)

배포 주소가 고정이라면 `.env.local` 또는 Vercel 환경 변수에 추가:

```
NEXT_PUBLIC_SITE_URL=https://실제배포주소.vercel.app
```

공유 시 사용하는 URL을 이 값으로 맞출 수 있습니다.

---

## 3. 변경된 공유 동작

- **표시 문구**: 설명 + 연애 스타일 일부 (최대 200자)로 길게 표시
- **버튼**:  
  - **결과 보기** → 현재 결과 페이지  
  - **나도 테스트하기** → 메인 페이지
- **링크**: `resultUrl`(결과 페이지 전체 URL)을 사용해 카드/버튼 클릭 시 해당 주소로 이동

도메인 등록까지 하면 공유 후 링크 연결이 정상적으로 동작합니다.
