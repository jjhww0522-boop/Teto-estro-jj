# 사이트 전수 점검 (Empty Corners) 체크리스트

애드센스 신청 전 tetolab.com(또는 배포 도메인)을 아래 기준으로 점검하세요.  
구글 봇은 **비어 있는 구석(Empty Corners)**을 매우 싫어합니다.

---

## 1. 404 페이지 확인

- [ ] **존재하지 않는 페이지로 연결되는 링크가 있는지 확인**
- **팁:** [drlinkcheck.com](https://www.drlinkcheck.com/) 등으로 전체 링크 스캔
- **현재 사이트 내부 링크:** `/`, `/test`, `/loading`, `/result`, `/result/[type]`, `/match`, `/privacy`, `/terms` — 위 경로는 모두 구현되어 있음
- **주의:** 레이아웃에서 OG 이미지로 `/images/og-main.png`를 참조합니다. **`public/images/og-main.png` 파일이 없으면 SNS/봇 요청 시 404가 날 수 있으므로**, 1200×630 이미지를 넣거나 `layout.tsx`의 `openGraph.images` 경로를 실제 존재하는 이미지로 바꾸세요.

---

## 2. 빈 버튼 및 플레이스홀더

- [ ] **"메뉴 준비 중"이나 내용이 없는 카테고리가 있으면 삭제하거나 임시 글이라도 채우기**
- **현재 상태:** `AdPlaceholder`는 "Advertisement", "이 공간은 광고 영역입니다" 문구로 **빈 칸이 아님**. 광고 승인 후 실제 광고 컴포넌트로 교체하면 됨.
- **권장:** 완전히 빈 버튼·메뉴·카테고리는 없도록 유지.

---

## 3. 이미지 Alt 태그

- [ ] **모든 `<img>`에 설명용 `alt` 넣기** (예: `alt="테토남 결과 이미지"`) — 시각 장애인 접근성·구글 봇 점수에 유리
- **현재 상태:** 결과/매치 페이지는 이모지로 유형을 표시하고, **본문에 `<img>` 요소는 없음.**  
  OG 이미지는 `layout.tsx`의 `openGraph.images[].alt`로 이미 지정됨 (`"테토 농도 분석기 대표 이미지"`).
- **추가 이미지를 넣을 때:** `<img>` 또는 `next/image` 사용 시 **반드시 `alt`** 를 넣으세요.

---

## 4. 반응형 레이아웃

- [ ] **아이폰, 갤럭시, 태블릿에서 메뉴 겹침·글자 잘림 없는지 확인**
- **확인 방법:** Chrome DevTools → 디바이스 툴바로 375px(아이폰), 360px(갤럭시), 768px(태블릿) 등으로 테스트
- **현재:** 루트 레이아웃에 `viewport` 메타가 반영되어 있으며, Tailwind로 반응형 클래스 사용 중. 실제 기기에서 한 번씩 눌러 보며 확인 권장.

---

## 5. 콘텐츠 양 (과학적 원리 텍스트)

- [ ] **메인 페이지 하단에 ‘과학적 원리’ 블로그 글이 텍스트로 잘 노출되는지 확인**
- **현재 구현:** 메인(`app/page.tsx`) 하단의 **연구 리포트** 섹션(`ResearchReport` 컴포넌트)에 Big Five, 애착 이론, 바넘 효과·인지적 정교화 등 **과학적 원리 설명이 전부 텍스트로 포함**되어 있음.  
  `<details>`로 접혀 있지만 **DOM에는 항상 존재**하므로 봇이 수집 가능.
- **권장:** 배포 후 메인 페이지에서 "📋 연구 리포트: 연애 성향 테스트, 과학일까?"를 펼쳐서 문단이 정상 노출되는지 확인.

---

## 점검 후 체크 요약

| 항목               | 점검 내용                          | 비고 |
|--------------------|------------------------------------|------|
| 404 확인           | 내부 링크 + `/images/og-main.png` | OG 이미지 파일 존재 확인 |
| 빈 버튼/플레이스홀더 | AdPlaceholder 외 빈 영역 없음      | 유지 |
| 이미지 Alt         | 현재 본문 `<img>` 없음, OG엔 alt 있음 | 추후 이미지 추가 시 alt 필수 |
| 반응형             | viewport 설정됨, 실제 기기 테스트 권장 | |
| 콘텐츠 양          | 연구 리포트에 과학 원리 텍스트 포함 | 펼쳐서 노출 확인 |

이 체크리스트를 모두 통과한 뒤 애드센스 신청을 진행하는 것을 권장합니다.
